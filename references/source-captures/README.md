# Source Captures

This directory stores local markdown captures fetched with `$baoyu-url-to-markdown`.

## Purpose

- create a reviewable local copy of public product pages and legal texts
- reduce dependence on live browsing during later planning
- support evidence-backed archetype matching in `online-reference-matrix.md`

## Usage Notes

- Prefer these local captures over memory when extracting verified signals.
- Check [../source-capture-qa.md](../source-capture-qa.md) before using a capture as a drafting baseline.
- Keep origin URLs in `online-reference-matrix.md` as the canonical public source.
- Do not treat a capture as target-product fact; captures are reference baselines only.
- If a capture looks too short, too generic, or obviously incomplete, re-fetch that page before relying on it.
